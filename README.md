# Этот проект создан исключительно для упрощения моей жизни при реселле мм2
## Как юзать
### Получение инвентаря
Запустите ваш эксплоит и выполните скрипт из файла get_inv_script.lua (работает только в мм2)

После успешного выполнения скрипта у вас появятся данные о вашем инвентаре в виде файла формата .txt
> На виндоус ищите файл по пути C:/Users/user/AppData/Local/your_executor/workspace/Inventory.txt

> С телефона или эмулятора ищите где то в файлах вашего эксплоита

После того как вы благополучно (надеюсь) нашли файл поменяйте формат файла на .json (просто замените .txt на .json) и перетащите его в src

Далее выполните main.py, должен появиться файл src/to_xlsx_format.txt, который можно легко преобразовать в таблицу простым разделением по столбцам по символу "_", а также в консоле будет выведена примерная итоговая валюта вашего инвентаря

Если нужна дополнительная настройка ссылок на суприм, с который скрипт будет брать валюты предметов, то добавляйте пару __редкость: ссылка__ в словарь links в файле src/utils/get_values.py
В main.py есть небольшой дебаг через консоль, который может вам упростить кастомизацию моего скрипта, он выводит данные о предметах, валюту которых не удалось определить
